2. Escala de Grises

Convertir una imagen a color en escala de grises.

Glosario:
  • BGR: Formato de color (Azul, Verde, Rojo)
  • cv2.cvtColor(): Convierte entre espacios de color
  • COLOR_BGR2GRAY: Convierte de BGR a escala de grises
  • NumPy Array: Matriz que representa la imagen
import cv2

img = cv2.imread('equipo.jpg')
# Devuelve una matriz NumPy en formato BGR (Azul, Verde, Rojo)

gris = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
# Convierte la imagen de color (BGR) a escala de grises

cv2.imshow('Original', img)
# Muestra una ventana con la imagen a color

cv2.imshow('Escala de Grises', gris)
# Muestra otra ventana con la imagen en escala de grises

cv2.waitKey(0)
# El par√°metro 0 significa "esperar indefinidamente"
# Si pusieras 1000, esperar√≠a 1 segundo antes de cerrar

cv2.destroyAllWindows()

fy=1.5: multiplica el alto por 1.5 (50% m√°s alto)
Resultado: img_grande es 1.5 veces m√°s grande
"""

# Mostrar las tres im√°genes

5. Recorte

Recortar una regi√≥n espec√≠fica de una imagen.

Glosario:
  • Slicing: [inicio:fin] para seleccionar regi√≥n
  • [y1:y2, x1:x2]: Coordenadas del recorte
  • ord(): Convierte car√°cter a c√≥digo ASCII
  • break: Sale del bucle while

# Cargar imagen

    recorte = gato[20:600, 50:600]  # Alto y ancho
    cv2.imshow('imagen original', gato)
    cv2.imshow('imagen recortada', recorte)
    
    tecla = cv2.waitKey(0)
    if tecla == ord('x'):
        cv2.destroyAllWindows()
        break

6. Rotaci√≥n

Rotar im√°genes con teclas de control.

Glosario:
  • cv2.rotate(): Rota la imagen
  • ROTATE_90_CLOCKWISE: Gira 90¬∞ a la derecha

7. Zoom Interactivo

Aplicar zoom con teclas + y - en la imagen.

Glosario:
  • zoom: Variable que controla el factor de escala
  • fx, fy: Factores de escala horizontal y vertical
  • + tecla: Aumenta el zoom 1.2x
  • - tecla: Reduce el zoom 0.5x
  • o tecla: Restaura el zoom original
def zoom():
    import cv2
    
    # Carga la imagen
    img_ori = cv2.imread("Spider.jpg")
    
    # Variable para el valor del zoom original
    zoom = 1.0
    
    while True:
        img = cv2.resize(img_ori, None, fx=zoom, fy=zoom)
        cv2.imshow("imagen nueva", img)
        tecla = cv2.waitKey(0)
        
        if tecla == ord('+'):
            zoom = zoom * 1.2
        elif tecla == ord('-'):
            zoom = zoom * 0.5
        elif tecla == ord('o'):
            zoom = 1.0
        elif tecla == ord('x'):
            break
    
    cv2.destroyAllWindows()

if __name__ == "__main__":
    zoom()

10. OCR para Placas

Reconocimiento de placas vehiculares con Tesseract.

Glosario:
  • pytesseract: Librer√≠a de OCR
  • tesseract_cmd: Ruta del ejecutable Tesseract
  • image_to_string(): Extrae texto de imagen
  • --psm 6: Modo de segmentaci√≥n de p√°gina
  • .strip(): Elimina espacios en blanco
  • .upper(): Convierte a may√∫sculas
import cv2
import pytesseract

# Configuraci√≥n de pytesseract
pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'

def leer_placas(imagenes, lista_placas_permitidas):
    placas_leidas = []
    placas_no_leidas = []
    
    for imagen_path in imagenes:
        # 1. Cargar la imagen
        img = cv2.imread(imagen_path)
        if img is None:
            print(f"Error: No se pudo cargar la imagen {imagen_path}")
            continue
        
        # 2. Extraer el texto de la placa
        try:
            texto_placa = pytesseract.image_to_string(img, config='--psm 6')
            texto_placa = texto_placa.strip().upper()
            print(f"Texto OCR detectado: {texto_placa}")
        except Exception as e:
            print(f"Error al extraer texto: {e}")
            placas_no_leidas.append(imagen_path)
            continue
        
        # 3. Verificar si est√° en la lista permitida
        if texto_placa in lista_placas_permitidas:
            placas_leidas.append(texto_placa)
        else:
            placas_no_leidas.append(texto_placa)
    
    # 4. Mostrar resultados
    if placas_leidas:
        print("Placas le√≠das:")
        for placa in placas_leidas:
            print(placa)
    else:
        print("No se encontraron placas le√≠das.")
    
    if placas_no_leidas:
        print("Placas no le√≠das o no permitidas:")
        for placa in placas_no_leidas:
            print(placa)

# Ejemplo de uso
imagenes_prueba = ["placa1.jpg", "placa2.jpg", "placa3.jpg", "placa4.jpg"]
lista_placas_permitidas = ["ABC1234", "XYZ5678", "DEF9012"]
leer_placas(imagenes_prueba, lista_placas_permitidas)
