3. An√°lisis de Sentimientos

üìö Glosario

  • Sentimiento: Polaridad emocional de un texto
  • tokens: Palabras individuales del texto
  • contador: Variable que acumula frecuencias
  • .split(): Separa string en lista de palabras
  • in: Operador para verificar pertenencia
palabraspositivas = ['feliz', 'contento', 'alegre', 'maravilloso', 'bonito', 'bonita'] palabrasnegativas = ['triste', 'solo', 'aburrido', 'malo', 'deprimido', 'horrible', 'muerte'] frase = "feliz, triste, triste" tokens = frase.replace(",", "").lower().split() contador_positiva = 0 contador_negativo = 0 for palabra in tokens: if palabra in palabraspositivas: contador_positiva += 1 elif palabra in palabrasnegativas: contador_negativo += 1 if contador_positiva > contador_negativo: sentimiento = "La frase es positiva" elif contador_negativo > contador_positiva: sentimiento = "La frase es negativa" print("Tokens:", tokens) print("Palabras positivas:", contador_positiva) print("Palabras negativas:", contador_negativo) print(sentimiento)
